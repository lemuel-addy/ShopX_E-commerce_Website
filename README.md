# ShopX_E-commerce_Website
ShopX is an Amazon clone E-commerce website built using JavaScript and React. It allows users to login and sign up, shop through items, add to Cart, remove from Cart and also calculates Subtotal based on items in Cart.
Deployed using Firebase to: https://shopx-e3796.web.app/

# Getting Started
To get started with ShopX, follow these steps:

Clone the repository:

### `git clone https://github.com/lemuel-addy/ShopX_E-commerce_Website.git`
Install the dependencies:

### `cd shopx`
### `npm install`
Start the development server:

### `npm start`
This will start the development server on http://localhost:3000. You should be able to see the ShopX home page in your web browser

# Features
ShopX includes the following features:

1. User login and signup using Firebase Authentication.
2. Add to Cart functionality.
3. Remove from Cart functionality.
4. Computation of subtotal based on items present in Cart.

# Technologies Used
ShopX is built using the following technologies:

1. ReactJS
2. Firebase
3. HTML and CSS
